Welcome

StandaloneHost is a simple application that can host one V2 Audio Unit soft synth. This is useful for playing soft synths that do not provide a standalone program, without starting up a DAW or other hosting app. It is essentially the Mac equivalent of the popular SAVIHost program for Windows with additional features.

This software is free, but if you find it useful and wish to support further development, please consider buying one or more of my $0.99 apps in the Mac App Store.
